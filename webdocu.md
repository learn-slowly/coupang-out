쿠팡의 사회적 문제는 개별 노동자, 소비자, 제보자의 삶을 파괴하는 구조적 폭력으로 요약된다.[1][2]

## 새벽배송 기사의 마지막 밤

50대 택배기사 '민수' 씨는 쿠팡 새벽배송으로 가족을 부양했다. 2025년 11월, 주 6일 밤 9시 출근해 다음날 새벽 7시까지 237개 물품을 배달하던 중 심장마비로 쓰러졌다. 식사도 휴식도 없이 "개처럼 뛰고 있다"고 호소한 지 보름 만이었다. 근로복지공단은 과로사를 인정했지만, 쿠팡은 소송으로 불복하며 유족에게 자료조차 제대로 주지 않았다.[1][3]

## 블랙리스트에 갇힌 물류센터 노동자

2022년, 쿠팡 물류센터에서 일하던 '지은' 씨는 노조 활동을 시작했다. 곧 재계약이 거부됐고, 1만6,450명 명단에 "노조원, 정상 업무 불가"로 적혔다. 다른 창고로 옮겨도 블랙리스트가 따라와 실직 상태가 길어졌다. 제보자가 이 리스트를 폭로하자 쿠팡은 제보자를 영업비밀 유출로 고소, 경찰이 압수수색까지 했다. 2025년 국회에서야 쿠팡이 "광범위 수집 죄송"하다 사과했지만, 지은 씨의 삶은 이미 무너졌다.[4][2][5]

## 짝퉁 속옷과 가짜 영양제의 피해자

'수진' 씨는 쿠팡에서 '캘빈 클라인' 속옷을 검색해 상단 상품을 샀다. 배송된 건 'Calwin Klein' 짝퉁이었다. 환불 소송 중 알게 된 건, 쿠팡 검색이 유사 상표·가짜를 우선 노출하는 구조였다. 비슷하게 40대 주부 '혜영' 씨는 '정품' 영양제를 사 먹다 성분 위조를 확인, 건강을 해친 뒤야 소송을 걸었다. 쿠팡은 "판매자 책임"이라며 플랫폼 면책을 주장했다.[6][7]

## 3,370만명의 그림자

2025년 11월, 4년 전 쿠팡을 탈퇴한 '철민' 씨에게 "쿠팡 배송 완료" 스미싱 문자가 왔다. 3,370만 계정 유출로 이름·전화번호가 피싱범 손에 넘어간 탓이었다. 천만원 피해 사례가 쏟아지자 방통위가 주의보를 냈지만, 쿠팡은 "조사 중"이라며 책임을 미뤘다. 철민 씨처럼 탈퇴자까지 피해를 본 건 개인정보 삭제 의무 위반의 증거였다.[8][9][2]

이 스토리들은 노동·소비자·정보 주체의 고통을 통해 쿠팡의 플랫폼 독점이 초래한 구조적 문제를 드러낸다. 개별 사례가 쌓여 사회적 참사로 번지는 과정이다.[10][11]

# 역할
너는 세계적인 수준의 Creative Developer야. Next.js, Framer Motion, Tailwind CSS를 마스터했어.

# 목표
`app/data/stories.ts`에 있는 데이터를 활용해서, 쿠팡의 사회적 문제를 고발하는 인터랙티브 웹 다큐멘터리 페이지를 만들어줘.

# 기술 스택
- Framework: Next.js 14+ (App Router)
- Styling: Tailwind CSS
- Animation: Framer Motion (필수)
- Icons: Lucide React

# 디자인 가이드
- 전체 테마: Deep Dark (#0a0a0a) 배경에 흰색 텍스트.
- 폰트: 가독성 좋은 Sans-serif (Inter 또는 Noto Sans KR).
- 분위기: 무겁고, 진지하며, 고발적인 톤앤매너.

# 구현해야 할 섹션 구조 (Scroll Section)
1. **Hero Section**:
   - 화면 중앙에 "편리함의 이면: 클릭의 비용"이라는 타이틀.
   - 스크롤을 내리면 화면이 어두워지며 Story 섹션으로 진입.

2. **Story Sections (반복)**:
   - `stories.ts` 데이터를 map으로 돌려서 렌더링.
   - 각 섹션은 `h-screen` (화면 꽉 참)이어야 하며 'Sticky' 포지셔닝을 활용해 스크롤텔링 구현.
   - **인터랙션(중요)**:
     - 텍스트는 스크롤에 따라 투명도(opacity)가 변하며 등장.
     - `stat.value` (숫자)는 스크롤이 해당 섹션에 도달했을 때 0부터 카운트업 되는 애니메이션 적용.
     - 배경에는 각 테마(theme)에 맞는 추상적인 도형이나 그라디언트가 은은하게 움직임.

3. **Outro Section**:
   - "이 불편한 진실을 공유해주세요" 메시지와 함께 다시 처음으로 돌아가는 버튼.

- 코드는 컴포넌트로 적절히 분리해줘 (예: `StoryCard.tsx`, `Hero.tsx`).
